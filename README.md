# Enterprise IT Incident Handling Portfolio

## Overview

This repository contains a small collection of simulated IT support incidents designed to demonstrate how first-line IT issues can be handled, documented, and escalated in an enterprise environment.

The focus of this project is not on advanced technical fixes, but on structured troubleshooting, clear communication, and safe decision-making when supporting business-critical systems.

## What This Project Demonstrates

- First-line IT incident handling
- Clear and consistent documentation
- Understanding of impact and urgency
- Appropriate escalation decisions
- Professional communication with users and senior IT staff

## Intended Audience

This project is intended for IT support, IT associate, and graduate IT roles in enterprise or business-critical environments.

## Structure

- `ticket-template.md`  
  Reusable template used to document each incident.

- `incidents/`  
  A set of completed example incidents using the template.

## Notes

All incidents in this repository are simulated for learning and demonstration purposes.

# Incident INC-001 - User Access Issue

## Incident ID

INC-001

## Issue Summary

User reports they are unable to log into SAP following recent location transfer.

## System/ Application

SAP, Windows 11

## User / Location

Office-based user recently transferred to a different location.

## Impact & Urgency

User is unable to access SAP to carry out daily tasks. Issue impacts productivity but does not affect muliple users.

## Initial Checks Performed

- Confirmed user identity and recent access changes
- Verified user account status was active and synchronised
- Checked for password or account lockout issue

## Actions Taken

- Reviewed user access permissions
- Identified missing SAP access follwoing location transfer
- Restored appropriate access permissions

## Escalation Decision

Resolved at first line. Access change was within role and permissions and did not require escalation.

## Resolution

User successfully logged into SAP after access group was restored.

## Post-Incident Notes/ Prevention

Recommend reviewing access group assignment during site transfer processes to prevent similar issues.

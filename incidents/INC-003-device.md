# Incident INC-003 - Device Performance Issue

## Incident ID

INC-003

## Issue Summary

User reports their Windows 11 device has become very slow during normal use.

## System / Application

Windows 11

## User / Location

Office based user

## Impact & Urgency

User experiencing slow performance from their office device impacting their daily work, single user impact with normal business priority.

## Initial Checks Performed

- Asked user when the performance issue started
- Confirm that the issue is isolated to a single user
- Verified user identity and checked for any recent changes
- Checked available disk space on the device
- Checked whether system updates were pending

## Actions Taken

- Guided user to restart device to apply any pending updates
- Advised user to close unused applications running in the background
- Confirmed performance improvement after restart

## Escalation Decision

Resolved at first line. Device performance returned to normal after applying pending updates, so no escalation was required.

## Resolution

User was able to resume normal use of their device and complete daily work tasks after pending updates were applied.

## Post-Incident Notes / Prevention

Advise users to restart their devices and apply updates regularly if device performance becomes slow.

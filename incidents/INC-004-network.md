# Incident INC-004 - Network Connectivity Issue

## Incident ID

INC-004

## Issue Summary

User reports they are unable to connect to a shared network resource from their workstation.

## System / Application

Windows 11, internal network resource

## User / Location

Office based user

## Impact & Urgency

User unable to access a shared network resource, impacting their ability to carry out daily tasks. Single-user issue with normal business priority.

## Initial Checks Performed

- Confirm that this issue is isolated to a single user
- Verified the users identity and for any recent changes
- Check the users account status is active and synchronised
- Checked whether the issue occurred on both Wi-Fi and wired connection
- Confirmed device was connected to the network
- Checked for any reported service outages

## Actions Taken

- Asked user to disconnect and reconnect to the network
- Guided user to restart the device to refresh network connection
- Asked user to retry access to the network resource after reconnecting

## Escaltion Decision

Resolved at first line. User was able to reconnect to the network after restarting the device, so no escalation was required.

## Resolution

User successfully regained access to the network resource and was able to resume normal work.

## Post-Incident Notes / Prevention

Advise users to reconnect to the network or restart their devices if network connectivity issues occur.

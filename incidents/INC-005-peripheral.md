# Incident INC-005 - Printer Issue

## Incident ID

INC-005

## Issue Summary

User reports they are unable to print documents to a shared office printer

## System / Application

Windows 11, network printer

## User / Location

Office based user

## Impact & Urgency

Single user issue with normal business priority. User unable to print documents required for daily work.

## Initial Checks Performed

- Confirmed issue was isolated to a single user
- Verified the printer is reachable on the network
- Verified the printer status for offline or error
- Confirmed the correct printer was set to default

## Actions Taken

- Restarted the printer to refresh connectivity
- Guided user to reconnect to the printer
- Confirm the user was printing to the correct default printer

## Escaltion Decision

Resolved at first line. After restarted the printer and reconnecting, the user was able to print successfully. No escalation was required.

## Resolution

User uccessfully printed documents using the shared office printer

## Post-Incident Notes / Prevention

Advise users to check default printer settings and restart the printer if printing issue occurs

# IT Incident Ticket Template

## Incident ID

INC-XXX

## Issue Summary

Brief description of the problem reported by the user.

## System / Application

System or application affected (e.g. Windows 11, SAP, Microsoft 365).

## User / Location

Who reported the issue and where they are based (office, plant, lab).

## Impact & Urgency

Describe how the issue affects the user or business and how urgent it is.

## Initial Checks Performed

List the first checks carried out before attempting any fix.

## Actions Taken

Steps taken to investigate or resolve the issue.

## Escalation Decision

State whether the issue was resolved at first line or escalated, and why.

## Resolution

Final outcome of the incident.

## Post-Incident Notes / Prevention

Any follow-up actions or steps to prevent recurrence.
